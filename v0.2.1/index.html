HomalgProject.jl

Introduction

HomalgProject.jl is an extension of GAP.jl providing simplified access to

  • homalg project,
  • CAP project.

These are open source GAP4 multi-package projects for constructive category theory and homological algebra with applications to module theory of commutative and non-commutative algebras and algebraic geometry.

Installation

julia> using Pkg
julia> Pkg.add("HomalgProject")
